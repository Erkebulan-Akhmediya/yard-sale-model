## Yard Sale Model

The repo contains two simulations of Yard Sale Model implemnted in Python and C.
I did this for educational purposes out of pure curiosity.

#### Getting Started
To run Python script:
```
python ysm.py
```

To compile C code (ignore warnings):
```
gcc -o ysm -O2 ysm.c
```

To run executable:
```
./ysm
```
